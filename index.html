Get started

A few minutes to set up.

Everything you need, nothing you don’t. Expand a step when you reach it.

Install the Adobe panel

Unzip the tester package, run the installer for your OS, and restart your host. No admin required.

🪟 Windows

  1. Unzip spielboard-tester.zip anywhere.
  2. Right-click install-gencanvas.ps1 → Run with PowerShell. If Windows blocks it: 
    powershell -ExecutionPolicy Bypass -File .\install-gencanvas.ps1
  3. Fully restart Premiere Pro / After Effects.
  4. Open Window → Extensions → Spielboard.

🍎 macOS

  1. Unzip the tester package anywhere.
  2. Double-click install-gencanvas.command (opens in Terminal). If Gatekeeper blocks it, right-click → Open, or: 
    xattr -d com.apple.quarantine ./install-gencanvas.command
  3. Fully quit & restart Premiere Pro / After Effects.
  4. Open Window → Extensions → Spielboard.
Install for DaVinci Resolve

Resolve uses a Workflow Integration plugin (not a ZXP), and it’s Studio-only on Windows for now.

Download spielboard-resolve.zip, unzip it, then quit Resolve completely (it locks plugin files while open) and run:

powershell -ExecutionPolicy Bypass -File .\install-resolve.ps1

Click Yes at the UAC prompt (it writes into ProgramData). Then restart Resolve → Workspace → Workflow Integrations → Spielboard.

macOS — coming soon. Resolve-for-Mac support is being finalized; on macOS use the Premiere / After Effects panel for now.

Add your fal.ai API key

Every generation runs on your fal.ai account, so you supply the key.

  1. Create a key at fal.ai/dashboard/keys and make sure your account has billing credit. (A fal key looks like key-id:secret.)
  2. In the Spielboard panel, click Settings (top-right).
  3. Paste the key into fal.ai API key and click Save.

Where your key lives

Stored locally only, readable just by you:

Windows
Documents\Spielboard\settings.json
macOS
~/Documents/Spielboard/settings.json

It’s never sent anywhere except fal.ai, and it’s never included in bug reports.

“Unverified extension” warning?

The alpha panel is self-signed with our own dev certificate — that’s expected. CEP checks a panel’s integrity, not who issued the cert, so Spielboard loads normally once installed. There’s nothing extra to “trust.”

System requirements

Host editor

  • Premiere Pro 2024+ (CEP panel) — Windows & macOS
  • After Effects 2024+ (CEP panel) — Windows & macOS
  • DaVinci Resolve Studio (Workflow Integration — the free edition can’t load it) — Windows; macOS coming soon

Platform

  • Windows 10 / 11 — Premiere / After Effects and Resolve Studio
  • macOS 12+ — Premiere / After Effects (Resolve for macOS coming soon)

Account & cost

  • A fal.ai API key with billing credit. Generations run on your key: image ≈ $0.01–0.08, video ≈ $0.50+ per run.
  • Optional: ffmpeg — only for “Pull frame / Pull clip” from the timeline.

The panel installs per-user and is self-signed for the alpha — no admin needed.
